# Library Capstone Project

A Django REST API for managing a library system. Users can browse, checkout, and return books with token-based authentication.

## Features

- User authentication (Token Auth)
- CRUD operations for books
- Book search and filter by title, author, ISBN, availability
- Checkout and return functionality
- Admin can track transactions
